# Meshtastic DIY

Meshtastic DIY 기기를 용도별로 분류하고 관리하기 위한 프로젝트입니다.

## 분류 체계

### 📱 휴대형 (Portable)
이동 중 사용을 위한 경량 배터리 기반 디바이스

**필수 옵션:**
- 배터리
- 디스플레이 (OLED / 전자잉크)
 - 스위치
**기타 옵션:**
- GPS
- 솔라패널
 - 키보드
### 🏠 거치형 (Desktop)
실내에서 고정 위치에 배치하는 상시 전원 연결 디바이스

**필수 옵션:**
- 디스플레이 (OLED / 전자잉크)

**기타 옵션:**
- 솔라패널
- 배터리
- 대형 화면 (MUI 사용 LCD)
 - 키보드

### 📡 설치형 (Installation)
옥외 또는 고정 설치를 위한 내구성 강화 디바이스

**필수 옵션:**
- 솔라패널
- 배터리
- 고이득 안테나

**기타 옵션:**
- 환경 센서

## 구조

각 카테고리별로 세부 모델과 설정을 관리합니다.

## 기타 설명

### 스위치 옵션 (휴대형)
휴대형 기기에서 사용 가능한 스위치:
- **일반 택트 스위치**: 기본 입력 방식
- **로터리 스위치**: 메쉬앱 설정 변경 필요
- **조이스틱**: 메쉬앱 설정 변경 필요

### 솔라패널 병렬 사용시 주의사항
역전압 방지 다이오드 사용 필수:
- **BAT54S**: 3.7V ~ 6V / 100mA 이하
- **SS14**: 6V ~ 12V / 0.5A 이하
